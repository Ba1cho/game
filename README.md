# exe
С помощью PyInstaller
PyInstaller позволяет создать exe-файл игры на Pygame. Некоторые шаги (для Windows): 
habr.com
Установить PyInstaller в командной строке: pip install pyinstaller.
Создать spec-файл (файл спецификации). Через командную строку в папке с проектом набрать: pyi-makespec --onefile mygame.py. Флаг --onefile создаёт файл спецификации, который позволит упаковать всё необходимое в один exe-файл.
Настроить spec-файл. В списке datas указываются файлы, которые нужно загрузить для работы приложения (изображения, музыка/звуки, шрифты). Каждый кортеж имеет два элемента строкового типа: первый — путь до файла, второй — имя папки для хранения файла во время выполнения программы ('.' — означает, что файл будет помещён во временную папку без подкаталога).
Создать exe в командной строке: pyinstaller mygame.spec. Exe-файл находится в папке dist.
